## 📚 背景

この資料は以下の目的で作成しました：

- 教師あり学習と教師なし学習の**違いの本質**を明確にする
- クラスタリング・次元削減といった基本手法の**直感的な理解**を促進する
- **PCAやt-SNE、k-means**など、頻出アルゴリズムの数式的・図解的解説
- 自己学習・後輩指導・の**解説資料としての活用**
